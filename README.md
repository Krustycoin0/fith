# fitness_app

- Flutter Fitness App.
- Contains Different Exercises.
- Clean Architecture.
- Bloc State Management.
- Api => retrofit.
- Local => floor.
- Injection Container => get_it.
- Responsive Design.
- Details Screen.
- Bookmarks.
- Gif Animation.

### Attention.
- Your Api Key from RapidApi is required.
